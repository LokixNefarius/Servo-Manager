Servo Manager
versão: 2.0

Instruções de uso:

- Adicione a imagem PNG, JPEG, JPG do seu servo astral com seu respectivo nome dentro da pasta "servos".
exemplo: Juht.png.

- o aplicativo irá gerar um arquivo com o nome do servo e informações como nome, saldo, código e tarefas.

- use o botão de adcionar código do servo, e insira o código numérico do seu servo astral.

- use um aplicativo gerador de frequência binaural caso o Servo Manager não reproduza o áudio. outra opção é pesquisar no Spotify ou baixar da internet.

- reproduza o Áudio, escreva o comando na caixa de texto do app, relaxe e olhe fixamente para a imagem do servo astral com seu respectivo sigilo. clique no botão de enviar.

- quando o servo realizar seu desejo, não se esqueça de entregar o código de recompensa usando o botão de alimentação.

- caso o servo escolhido tenha uma função contínua, você pode definir a alimentação por meio do código, alimentando o servo diariamente, semanalmente ou mensalmente.


sobre o código de cada servo astral:

- servos astrais podem ser linkados a numeros por analogia numerológica. cada número pode estar atribuído a habilidades e função do servidor.

- pode-se criar um código numérico baseado em uma frase de afirmação correspondente ao servo ou mantra, de acordo com o significado dos números.
exemplo: A sorte está comigo.
código: 321
 - 3("sorte"), 
 - 2("estar junto"), 
 - 1("comigo", "eu", "self")
